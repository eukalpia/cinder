Cinder provides a testing framework for TUI components. This guide shows you how to write tests.

## Test setup

Import the test library:

```dart
import 'package:cinder/cinder.dart';
import 'package:cinder/cinder_test.dart';
import 'package:test/test.dart';

void main() {
  test('counter increments', () async {
    await testCinder(
      'counter test',
      (tester) async {
        await tester.pumpWidget(Counter());

        expect(tester.terminalState, containsText('Count: 0'));

        await tester.sendKey(LogicalKey.space);

        expect(tester.terminalState, containsText('Count: 1'));
      },
    );
  });
}
```

## testCinder function

`testCinder` runs a test with a virtual terminal:

```dart
await testCinder(
  'test description',
  (tester) async {
    // Test code here
  },
  size: Size(80, 24),  // Optional terminal size
  debugPrintAfterPump: false,  // Optional debug output
);
```

## CinderTester

The tester provides methods for testing:

```dart
// Render a widget
await tester.pumpWidget(MyWidget());

// Send a key press
await tester.sendKey(LogicalKey.enter);

// Send multiple keys
await tester.sendKeys([
  LogicalKey.keyH,
  LogicalKey.keyE,
  LogicalKey.keyL,
  LogicalKey.keyL,
  LogicalKey.keyO,
]);

// Type text
await tester.enterText('hello');

// Access terminal state
final state = tester.terminalState;
```

## Basic assertions

Use matchers to verify output:

```dart
test('text display', () async {
  await testCinder(
    'displays text',
    (tester) async {
      await tester.pumpWidget(Text('Hello'));

      // Check if text is present
      expect(tester.terminalState, containsText('Hello'));
    },
  );
});
```

### containsText

Check if text appears anywhere:

```dart
expect(tester.terminalState, containsText('Hello'));
```

### hasTextAt

Check text at specific position:

```dart
expect(tester.terminalState, hasTextAt(5, 10, 'Title'));
```

### hasStyledText

Check text with specific styling:

```dart
expect(
  tester.terminalState,
  hasStyledText('Error', color: Colors.red),
);
```

## Testing interactions

Simulate user input:

```dart
test('button click', () async {
  await testCinder(
    'handles click',
    (tester) async {
      var clicked = false;

      await tester.pumpWidget(
        Focusable(
          focused: true,
          onKeyEvent: (event) {
            if (event.logicalKey == LogicalKey.enter) {
              clicked = true;
              return true;
            }
            return false;
          },
          child: Text('Press Enter'),
        ),
      );

      await tester.sendKey(LogicalKey.enter);

      expect(clicked, isTrue);
    },
  );
});
```

### Simulating keyboard input

```dart
// Single key
await tester.sendKey(LogicalKey.arrowDown);

// Multiple keys
await tester.sendKeys([
  LogicalKey.keyC,
  LogicalKey.keyT,
  LogicalKey.keyR,
  LogicalKey.keyL,
]);

// Type text
await tester.enterText('hello world');

// With modifiers
await tester.sendKey(
  LogicalKey.keyS,
  control: true,  // Ctrl+S
);
```

## Testing state changes

Test components that change over time:

```dart
test('counter state', () async {
  await testCinder(
    'counter increments',
    (tester) async {
      await tester.pumpWidget(Counter());

      expect(tester.terminalState, containsText('Count: 0'));

      await tester.sendKey(LogicalKey.space);
      expect(tester.terminalState, containsText('Count: 1'));

      await tester.sendKey(LogicalKey.space);
      expect(tester.terminalState, containsText('Count: 2'));
    },
  );
});
```

## Best practices

### Test one thing at a time

```dart
// Good - focused test
test('displays title', () async {
  await testCinder('title', (tester) async {
    await tester.pumpWidget(App());
    expect(tester.terminalState, containsText('My App'));
  });
});

// Bad - testing too much
test('everything', () async {
  // Testing multiple unrelated things
});
```

### Use descriptive test names

```dart
// Good
test('increments counter when space pressed', () async { ... });

// Bad
test('counter', () async { ... });
```

### Test edge cases

```dart
test('handles empty list', () async { ... });
test('handles very long text', () async { ... });
test('handles terminal resize', () async { ... });
```

### Clean up resources

```dart
test('disposes controllers', () async {
  await testCinder('disposal', (tester) async {
    final controller = TextEditingController();
    await tester.pumpWidget(TextField(controller: controller));

    // Widget should dispose controller
    controller.dispose();  // Should not throw
  });
});
```
